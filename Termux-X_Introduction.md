# Termux-X 项目介绍与近期更新说明

## 1. Termux-X 简介

**Termux-X** 是一款基于 **ZeroTermux** 进行二次开发（二开）的增强版终端模拟器应用。它保留了 ZeroTermux 的强大功能，并在此基础上进行了深度定制与优化，旨在为移动端渗透测试人员和极客提供更便捷、更强大的操作环境。

### 核心亮点
*   **一键免 Root 运行 Kali NetHunter**：这是 Termux-X 最具杀伤力的功能。用户无需对手机进行 Root 操作，即可通过内置的一键脚本/功能，快速启动并运行完整的 Kali NetHunter 渗透测试环境，极大降低了移动安全审计的门槛。
*   **深度定制 UI**：全新的 Material Design 风格图标与界面设计，不仅美观，更符合操作直觉。
*   **AI 智能助手集成**：内置 AI 辅助功能，支持自然语言转 Shell 命令，不仅能解释报错，还能辅助生成 Payload 和进行代码审计。
*   **丰富工具箱**：预装并集成了 Metasploit, Sqlmap, Seeker 等常用黑客工具的快捷入口。

## 3. 近期更新与修改内容 (App端)

针对 **Termux-X** 客户端应用进行了深度的功能增强与体验优化，主要包含以下方面：

### 🛠️ 品牌与界面重塑
*   **全局品牌更名**：应用名称正式变更为 **Termux-X**。已完成启动页、设置菜单、关于页面及主界面标题的文本替换，强化品牌识别度。
*   **UI 布局调整**：
    *   **版块重组**：调整了主页功能版块的顺序，将 **X11 功能区** 移动至 **Kali NetHunter** 版块下方，**常用功能**上方，优化了视觉层级。
    *   **界面精简**：移除了 X11 功能区中冗余的“安装X11”按钮，并对剩余功能按钮进行了重新排版，使布局更平衡。

### �️ 桌面环境支持
*   **Termux 原生图形化界面**：
    *   **一键启动**：深度集成了 **Termux-X11** 与 **XFCE4** 桌面环境。用户只需点击“启动桌面”，即可自动配置 X11 服务、环境变量并启动 XFCE4 桌面，无需繁琐的手动输入命令。
    *   **智能联动**：自动唤起 Termux-X11 应用，实现从命令行到图形界面的无缝切换。
*   **Kali NetHunter 图形化桌面**：
    *   **一键 KeX 连接**：内置对 **NetHunter KeX** 的支持。点击“启动图形化”即可后台启动 KeX 服务并自动跳转至 NetHunter KeX 客户端，实现 Kali Linux 桌面的即时访问。
    *   **依赖检测**：智能检测 NetHunter KeX 客户端安装状态，若未安装将自动引导下载。

### �🚀 功能增强
*   **会话管理升级**：
    *   重构了“会话”按钮交互，新增 **PopupMenu** 菜单。
    *   支持快捷选择“新建 Termux Shell”或直接进入“Kali Shell”会话。

### ⚡ 稳定性与 Bug 修复
*   **X11 桌面启动优化**：
    *   **解决“点击两次”问题**：修复了启动脚本中的竞态条件（Race Condition），增加 3秒 等待机制，确保 X Server 完全就绪后再加载 XFCE4 桌面。
    *   **容错机制**：优化 Shell 命令连接符（由 `&&` 改为 `;`），确保即使某个子命令（如 export display）警告，后续启动命令仍能继续执行。
*   **进程管理修复**：
    *   **彻底停止桌面**：重写了“停止桌面”的命令逻辑，使用 `pkill -f com.termux.x11` 精准查杀进程，解决了部分环境残留导致无法重新启动的问题。
*   **崩溃修复**：修复了主页初始化时的空指针异常（NullPointerException），增强了应用稳定性。
